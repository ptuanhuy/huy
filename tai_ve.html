Kho Công Cụ Kỹ Thuật

Danh sách 17 nhóm tiện ích . Dù bạn dùng Windows hay macOS, mình đều có giải pháp phần mềm tốt nhất.

Rất tiếc, chưa có phần mềm nào khớp với tìm kiếm của bạn.
WINDOWS
1. Giải nén (.exe)

7-Zip & WinRAR

Bảo bối "quốc dân" đọc trọn vẹn file RAR, ZIP, 7Z, ISO. Phần mềm bắt buộc phải cài ngay sau khi cài Win.

Tải 7-Zip (.exe) Tải WinRAR (.exe)
2. Đồ họa / Văn bản

Bộ Font Tiếng Việt (Win)

Tổng hợp 2000+ Font Việt hóa (VNI, TCVN3, SVN, UTM). Cài đặt dễ dàng vào thẳng thư mục Fonts của Windows.

Tải trọn bộ Font Win
Cách cài Font trên Windows

Giải nén thư mục, bôi đen toàn bộ Font, click chuột phải chọn Install for all users.

3. Download Tốc Độ

Internet Download Manager (IDM)

Phần mềm IDM huyền thoại độc quyền cho Windows. Tự động chia nhỏ luồng tải file và bắt link video cực nhanh.

Tải IDM (.exe)
4. Tìm Kiếm Siêu Tốc

Everything (Voidtools)

Tìm kiếm file ẩn, thư mục trên ổ cứng NTFS của Windows với tốc độ tính bằng mili-giây, nhanh hơn 1000 lần Search mặc định.

Tải Everything (.exe)
5. Văn phòng / PDF

Foxit PDF Reader

Trình đọc, ghi chú và in ấn file PDF cực kỳ nhẹ. Không ngốn RAM và CPU hệ thống Windows như Acrobat.

Tải Foxit Reader (.exe)
6. Code / Text Editor

Notepad++

Trình soạn thảo mã nguồn nhẹ nhất cho Windows. Hỗ trợ đa ngôn ngữ, tự động lưu nháp chống mất điện.

Tải Notepad++ (.exe)
7. Hỗ trợ từ xa

UltraViewer / AnyDesk

Bản cài đặt dành cho Windows. Cài đặt để kỹ thuật viên kết nối và sửa lỗi phần mềm cho bạn ngay lập tức.

Tải UltraViewer (.exe) Tải AnyDesk (.exe)
8. Ngôn ngữ

Sogou Pinyin (Win)

Bộ gõ tiếng Trung số 1 hiện nay cho Windows. Tích hợp AI gợi ý từ vựng cực kỳ chuẩn xác.

Tải Sogou (.exe)
9. Tối ưu năng suất

Microsoft PowerToys

Bộ công cụ "chính chủ" từ Microsoft. Giúp ghim cửa sổ trên cùng, chia màn hình thông minh cho Win 10/11.

Tải PowerToys (.exe)
10. Quản lý Ổ Đĩa

Rufus (Tạo USB Boot)

Công cụ Portable định dạng USB chuyên dụng trên Win. Dùng để burn file ISO cài đặt Windows hoặc WinPE cứu hộ.

Tải Rufus (.exe)
11. Dọn dẹp rác

Revo Uninstaller

Chuyên trị phần mềm "cứng đầu" trên Win. Gỡ ứng dụng tận gốc rễ, quét sạch khóa Registry còn thừa.

Tải Revo (.exe)
12. Hardware

CrystalDiskInfo & CPU-Z

Đọc thông số nội tạng máy tính Windows. Test tốc độ, tuổi thọ ổ cứng (SSD/HDD) và đo nhiệt độ, thông số RAM, CPU.

Check Ổ Cứng (.exe) CPU-Z (.exe)
13. Bảo mật hệ thống

Kaspersky Standard (Win)

Trình diệt virus lừng danh cho Windows. Cấu hình nhẹ, bảo vệ hệ thống khỏi ransomware mã hóa dữ liệu.

Tải Kaspersky (.exe)
14. Lệnh Hệ Thống

Tổng hợp Lệnh RUN

Bí kíp 100+ lệnh (Win + R) mở nhanh các thiết lập ẩn sâu bên trong Windows dành riêng cho kỹ thuật viên.

Xem lệnh Windows
15. Kỹ thuật / CAD

Bộ Font AutoCAD (.shx)

Tổng hợp 2000+ Font SHX và TTF chuyên dụng cho dân xây dựng, cơ khí. Khắc phục 100% lỗi font bản vẽ CAD.

Tải trọn bộ Font CAD
Cách cài Font CAD

Copy toàn bộ file .shx dán vào thư mục C:\Program Files\Autodesk\AutoCAD 20xx\Fonts.

16. Kế toán / Thuế

HTKK & iTaxViewer

Bộ phần mềm Hỗ trợ kê khai thuế và đọc tờ khai định dạng XML chuẩn nhất từ Tổng cục Thuế Việt Nam.

Tải HTKK Tải iTaxViewer
17. Máy In / Photocopy

Driver Canon, Ricoh, HP

Bộ cài đặt Driver chuẩn cho các dòng máy in, máy photocopy phổ biến tại văn phòng. Hỗ trợ in qua mạng LAN (IP).

Driver Canon Driver Ricoh Driver HP
macOS
1. Giải nén (.dmg)

The Unarchiver / Keka

Tiện ích giải nén chuyên dụng của Mac. Hạn chế tối đa lỗi font tiếng Việt khi giải nén file RAR/ZIP từ Win gửi qua.

Tải The Unarchiver (.dmg) Tải Keka (.dmg)
2. Đồ họa / Văn bản

Bộ Font Tiếng Việt (Mac)

Bộ sưu tập Font việt hóa (SVN, UTM) đã được fix lỗi hiển thị trên hệ điều hành macOS, quản lý qua Font Book của Mac.

Tải trọn bộ Font Mac
Cách cài Font trên Mac

Mở Font Book (Sổ Phông chữ) -> Bấm dấu + và chọn thư mục font vừa giải nén để import.

3. Download Tốc Độ

Free Download Manager (FDM)

Do IDM không hỗ trợ Mac, FDM là giải pháp thay thế hoàn hảo nhất. Tăng tốc tải file, bắt link cực ngon trên Safari và Chrome.

Tải FDM (.dmg)
4. Tìm Kiếm Siêu Tốc

Raycast / Alfred

Ứng dụng thay thế Spotlight (Cmd + Space) thần thánh. Quản lý clipboard, tính toán, chuyển đổi tiền tệ siêu nhanh.

Tải Raycast (.dmg)
5. Văn phòng / PDF

Foxit Reader / PDF Expert

Trình xem và chỉnh sửa PDF cao cấp dành riêng cho Mac. Mượt mà, hỗ trợ ký tên điện tử nhanh chóng.

Tải Foxit Reader (.dmg)
6. Code / Text Editor

Sublime Text

Vì Mac không có Notepad++, Sublime Text là lựa chọn số 1. Cực kỳ gọn nhẹ, khởi động tức thì, màu code đẹp.

Tải Sublime Text (.dmg)
7. Hỗ trợ từ xa

AnyDesk / TeamViewer

Khuyên dùng AnyDesk trên Mac vì độ trễ thấp. Yêu cầu cấp quyền Screen Recording & Accessibility trong System Settings để hoạt động.

Tải AnyDesk (.dmg) Tải TeamViewer (.dmg)
8. Ngôn ngữ

Sogou Pinyin (Mac)

Bộ gõ tiếng Trung được tối ưu riêng biệt trên chip Apple Silicon (M1/M2/M3) và Intel. Trải nghiệm gõ mượt mà.

Tải Sogou (.dmg)
9. Tối ưu năng suất

Rectangle & Maccy

Hai tiện ích thiết yếu: Rectangle giúp chia đôi, chia tư cửa sổ như Win. Maccy quản lý lịch sử Copy/Paste tuyệt đỉnh.

Tải Rectangle (.dmg) Tải Maccy (.dmg)
10. Quản lý Ổ Đĩa

Paragon NTFS & BalenaEtcher

Paragon giúp Mac đọc/ghi trực tiếp USB NTFS của Windows. BalenaEtcher là phần mềm burn USB Boot số 1 trên Mac.

Tải Paragon NTFS (.dmg) Tải BalenaEtcher (.dmg)
11. Dọn dẹp rác

AppCleaner / CleanMyMac X

Kéo app thả vào đây để gỡ sạch sẽ 100% rác cache trên Mac. Giải phóng hàng chục GB dung lượng cho ổ SSD.

Tải AppCleaner (Free) Web CleanMyMac X
12. Hardware

iStat Menus / TG Pro

Theo dõi thông số kỹ thuật ngay trên thanh Menu bar của Mac: Nhiệt độ CPU, tốc độ quạt gió, RAM và tình trạng Pin.

Xem iStat Menus (.dmg)
13. Bảo mật hệ thống

Kaspersky Standard (Mac)

Bảo vệ Mac khỏi các phần mềm đào coin ẩn danh, chặn virus lây lan sang các máy Windows khác khi cắm USB.

Tải Kaspersky (.dmg)
14. Lệnh Hệ Thống

Lệnh Terminal & Phím Tắt

Cẩm nang các câu lệnh Terminal để can thiệp hệ thống macOS (như tắt Gatekeeper, hiển thị file ẩn) và các tổ hợp phím tắt.

Xem Phím tắt Apple
15. Kỹ thuật / CAD

Bộ Font AutoCAD (Mac)

Bộ Font SHX chuyên dụng giúp dân kỹ thuật mở bản vẽ AutoCAD trên MacBook không bị lỗi font chữ tiếng Việt.

Tải trọn bộ Font CAD
Cách cài Font CAD (Mac)

Chuột phải vào app AutoCAD -> Show Package Contents -> Contents/Resources/Fonts/shx và dán font vào.

16. Máy ảo / Kế toán

Parallels Desktop (Chạy Win)

Giải pháp số 1 để dân Kế toán dùng MacBook có thể chạy phần mềm thuế HTKK, iTaxViewer mượt mà như máy Windows.

Tải Parallels Desktop (.dmg)
17. Máy In / Photocopy

Driver Máy In cho macOS

Bộ Driver hỗ trợ in ấn từ MacBook ra các dòng máy in, máy photocopy văn phòng (Canon, Ricoh). Khắc phục lỗi in không ra giấy.

Driver Canon (Mac) Driver Ricoh (Mac) Driver HP (Mac)
